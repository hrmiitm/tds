## Q2: Why can't I access my local server from another device?

**Answer:** Multiple reasons:

1. **Binding to localhost** - Server only listens on 127.0.0.1
2. **Firewall** - Blocking incoming connections
3. **NAT** - Your router doesn't forward traffic

**Solutions:**
- Use `--host 0.0.0.0` to bind to all interfaces
- Open port in firewall
- Use Ngrok for public access
